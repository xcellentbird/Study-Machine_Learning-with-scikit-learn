## Django 설치
1. venv 가상 환경 만들기
```
$ python3 -m venv dj-preview
$ source bin/activate
```

2. django 설치
![image](https://user-images.githubusercontent.com/59414764/124951984-d2954180-e04e-11eb-902c-616d2bb281e0.png)

3. 프로젝트 생성
```
$ django-admin startproject <project_name>
$ cd <project_name>
$ python3 manage.py runserver
```
